# azul-prolog-sim

For 4 players Azul Game in Prolog

### Prerequisites

**Unix**
* Run the next command:
```
sudo apt-get install swi-prolog
```
**Windows**

* Download and execute *swipl-8.0.3-1.x(64/32).exe*

### Download and Run
* Download `main.pl`.
* Run:
```
$ swipl --no-top-level main.pl
```
